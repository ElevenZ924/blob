# 低代码平台简介

## 什么是低代码平台

低代码平台是一种软件平台，它允许用户通过可视化界面来创建和部署应用程序，而无需编写代码。低代码平台通常提供了一种简单易用的方式来构建应用程序，而无需花费大量时间和精力来编写代码。

## VTJ.PRO

[VTJ.PRO](https://www.vtj.pro/) 是一个低代码平台，它提供了一种简单易用的方式来构建应用程序。
