## makeself 

简介: [makeself](https://makeself.io/) 一个用于 Unix 系统的自提取存档工具, 可以从一个目录生成自解压的压缩 tar 归档文件。
生成的文件看起来像是一个 shell 脚本（很多这样的文件有.run 后缀），可以直接运行。然后归档文件会解压到临时目录，并执行一个可选的任意命令（例如安装脚本）。
